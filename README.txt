== Description ==

The plugin creates fields for contact details on the site.
It can be used both for a business card site and for a company site.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `allcontactorgv.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy
